JS, o JavaScript, es un lenguaje de programación para hacer páginas web interactivas y dinámicas. Se utiliza para crear animaciones, validar formularios, controlar multimedia y permitir la actualización de contenido sin recargar la página. Su uso principal es en el desarrollo web, pero también se utiliza para aplicaciones móviles y de escritorio, y se puede ejecutar tanto en el navegador (lado del cliente) como en servidores (lado del servidor) con herramientas como Node.js.
